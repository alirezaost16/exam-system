Sequence Diagram
نمودار توالی — جریان شرکت در آزمون
👤 دانشجو 🌐 Browser ⚡ FastAPI 🗄 Database
کلیک روی آزمون GET /exams/{id}/take →
بررسی session → query user_sessions
← user object
بررسی آزمون فعال → query exams + questions
← exam + questions
← HTML صفحه آزمون
── loop: دانشجو به سوالات پاسخ می‌دهد (auto-save to localStorage) ──
کلیک «ارسال» POST /exams/{id}/submit →
تصحیح خودکار → INSERT exam_results
← result_id
← Redirect /results/{id}
مشاهده نتیجه GET /results/{id} →
load result → query exam_results + questions
← صفحه نتیجه + نمره
نکات کلیدی:
✦ اگر session منقضی شده باشد → Redirect به /login
✦ اگر آزمون خارج از بازه زمانی باشد → صفحه exam_closed
✦ اگر دانشجو قبلاً شرکت کرده → Redirect مستقیم به نتیجه
✦ اگر وقت تمام شد → JS به‌صورت خودکار فرم را submit می‌کند
